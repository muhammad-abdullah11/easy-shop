# Easy Shop

A modern e-commerce application built with React, Vite, and React Router.

## Features

- **Dynamic Navigation:** Seamless browsing between Men, Women, and Juniors sections.
- **Category Pages:** Dedicated landing pages for each main category featuring promotional banners and curated sections.
- **Product Collections:** Filterable and sortable grids for browsing products by type (Denim, Footwear, T-Shirts, etc.).
- **Product Details:** Detailed product views with image galleries, size selection, and description.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop devices.
- **Search Functionality:** Find products quickly.
- **Cart Integration:** (Visual placeholder for cart functionality).

## Tech Stack

- **Frontend Framework:** React 18
- **Build Tool:** Vite
- **Routing:** React Router DOM
- **Styling:** Tailwind CSS
- **Icons:** React Icons

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Run the development server:**
    ```bash
    npm run dev
    ```

3.  **Build for production:**
    ```bash
    npm run build
    ```

## Project Structure

- `src/Components/Pages`: Main landing pages (HomePage).
- `src/Components`: Reusable components (Header, Footer, Product, Collections, CategoryPage).
- `src/assets`: JSON data files for products and banners.
